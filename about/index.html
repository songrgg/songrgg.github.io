我是谁

在杭州电子科技大学被初始化, 于魔都运行.

会?

计算机方面真的是杂学, 心路旅程复杂.

  • 看完CSAPP, 略看SICP, APUE, 龙书, 算法导论. 大学里实现过一些系统的雏形, 像HTTP服务器, scheme解释器什么的. 也做了200+算法题, 实现过常用的数据结构.
  • Web开发从C# ASP -> Java Struts -> Php CodeIgniter, 前端也就会懂些皮毛, 能用css+html+js实现给定的需求, 不过凭借着对解释器实现的经历, 写js竟然没什么不习惯, 还玩上了angularJS.
  • Mysql从表结构设计到索引优化, 慢查询分析.
  • 略看过一些分布式系统的论文, Bigtable, GFS, Map-Reduce, 正跟着MIT的好老师们用golang实现一个fault-tolerant的分布式系统.

爱好

学习基础架构, 醉心于可以一直cool下去的路. 想玩的很多, 目前是跟完MIT-6.824, 完善一个成熟的scheme解释器(按照sheme规格文档所述). 解决一些在工作中遇到的有挑战的事…

本质
