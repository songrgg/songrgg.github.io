闭包this绑定

新增Closure::call，优化Closure::bindTo(JavaScript中bind，apply也是一样的应用场景)。

2-3行新建Foo的对象，第4行创建了一个foobar的闭包，第5行调用闭包的call方法，将闭包体中的$this动态绑定到$foo并执行。

同时官网上进行性能测试，Closure::call的性能优于Closure::bindTo。
